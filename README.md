## Image Analyzer

### usage

```
Usage: image_analyzer.rb [these options are necessary]
      --start N                    Search at the Nth result and save in the Nth filename
      --size N                     Search and save N images

```

### Dependencies
1. ruby gems
   1. google-search
   2. tesseract-ocr (interface for c program)
2. c programs
   1. tesseract-ocr
