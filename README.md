## Image Analyzer

### usage

```
ruby bin/image_analyzer [start_idx] [count] 
     # start_idx is the start number of the image file and search query
     # count is the number of images to download
```

### Dependencies
1. ruby gems
   1. google-search
   2. tesseract-ocr (interface for c program)
2. c programs
   1. tesseract-ocr
